LUYỆN TẬP

1. Phần tử HTML có thể ẩn đi trên trang web được không? Nếu có thì dùng lệnh CSS gì?

Có, có thể sử dụng CSS để ẩn phần tử HTML bằng các cách sau:

display: none; /* Ẩn phần tử và không chiếm không gian */
visibility: hidden; /* Ẩn phần tử nhưng vẫn chiếm không gian */

2. Giải thích ý nghĩa định dạng sau:

.test .test_more {background-color: red;}

Định dạng trên áp dụng màu nền đỏ cho bất kỳ phần tử nào có class "test_more" nhưng chỉ khi nó nằm bên trong một phần tử có class "test".

VẬN DỤNG

1. Định dạng các đoạn văn quan trọng

Ta có thể sử dụng CSS để định dạng riêng các đoạn văn quan trọng bằng cách:

#P1 { color: blue; font-weight: bold; }
#P2 { color: green; font-style: italic; }
#P3 { color: red; text-decoration: underline; }

2. Định dạng các khung với thông số khác nhau

Ta có thể thiết lập viền khung khác nhau bằng cách:

.box1 { border: 2px solid black; }
.box2 { border: 3px dashed blue; }
.box3 { border: 4px dotted red; }
